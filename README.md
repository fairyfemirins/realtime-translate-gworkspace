# RealTimeTranslate for Google Workspace

A Google Apps Script add-on for real-time translation in Google Docs, Sheets, and Slides using LibreTranslate.

## Features
- Translate selected text in real-time.
- Auto-detect source language.
- Supports multiple languages.
- Free and open-source.

## Installation
1. Open a Google Doc.
2. Click **Extensions > Apps Script**.
3. Paste the contents of `Code.js` and `Sidebar.html` into the script editor.
4. Save and reload the document.
5. The **RealTimeTranslate** menu will appear in the toolbar.

## Usage
1. Select text in your document.
2. Click **Extensions > RealTimeTranslate > Translate Selection**.
3. Choose the source and target languages.
4. Click **Translate**.
5. The translated text will appear in the sidebar.

## Supported Languages
- English (en)
- Spanish (es)
- French (fr)
- German (de)
- Chinese (zh)
- Japanese (ja)
